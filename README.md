## Installation

1. Clone the repository:

```bash
git clone https://github.com/Srinath10X/catppuccin-waybar.git
```

2. Navigate to the project directory:

```bash
cd catppuccin-waybar/waybar
```

3. Copy the config to your .config folder

```bash
make install
```

## Usage

1. Run Waybar:

```bash
waybar
```
